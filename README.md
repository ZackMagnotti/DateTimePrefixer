# DateTimePrefixer
---------------------------------------------------------------
DateTimePrefixer

Created By Zachary Magnotti

A script which reads file metadata from a library of images, and
copys them into a new folderwith the Date and Time prefixed 
to the file name.

Useful for sorting pictures chronologically
----------------------------------------------------------------

Command line arguments
    --input-folder folder  # folder containing images
    --output-folder folder # folder for output images, creates new folder if needed
